Dashboard Layout

The new React UI needs to fit: metrics, pipeline triggers, video queue, activity log, cost tracker, and channel health. Which structure feels right?

A

Command Bar + Full-Width Queue

Sticky header, then a compact command strip (metrics + action buttons), then the video queue takes the full width. Activity/cost/health collapse into a bottom drawer or are accessible via small inline toggles.

B

Two-Panel (Sidebar + Queue)

Left sidebar (fixed, ~280px) holds metrics, pipeline stage counts, action buttons, channel health, and cost tracker. Right panel is the scrollable video queue.

C

Stacked Sections (Full-Width)

Everything full-width and stacked: stats row → pipeline kanban → video queue → activity + stats. User scrolls down to see all. Queue is in the middle, always above the fold on load.
